What is the Documents Object Model (DOM)?

DOM is a tree like structure and each element represent as an object of the tree. And it's interact with the document contents.

How do you select an element from the DOM?

There are multiple way to select an element from the DOM.

  • document.getElementByID('ID')
  • document.getElementsByClassName('className')
  • document.getElementsByTagName('tagName')
  • document.querySelector('selector')
  • document.querySelectorAll('selector')

What is event delegation in the context of the DOM, and is it useful?

Event Delegation is a technique in the DOM where a event listener is added in the parent element to manage of it's child elements, even through child elements added dynamically

Yes it is useful, when many child need to be handle or when child are added dynamically and optimize the performance and reduce memory space.

How do you manipulation an element's attributes and styles using the DOM?

There are many way to manipulation an elements attributes. We can add, modify or remove attributes of an element using the DOM for example - setAttribute(attributeName, value), getAttribute(attributeName), removeAttribute(attributeName), hasAttribute(attributeName)

We can manipulate styles using the style property or by adding/removing classes for example - classList.add(), classList.remove(), classList.contains(),
